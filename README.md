# Tic-Tac-Toe

Simple tic-tac-toe game, developed in javascript.
  
## Usage
in cmd, choose a folder and run the command:
 ```bash
  git clone https://github.com/brunorp/Tic-tac-toe.git 
 ``` 

 then open the project folder and open src/index.html file.
